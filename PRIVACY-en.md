# Privacy Policy

Last updated: `2026-05-20`

X/Twitter Following Manager is a browser extension for X/Twitter on the web. It shows account following and follower counts directly on the page, and provides local tools for organizing, importing, exporting, and refreshing account records.

## Information We Process

While the extension is running, it may process the following information locally in your browser:

- X/Twitter account display names, handles, user IDs, and profile URLs
- account following counts, follower counts, bios, websites, and avatar information
- latest browsed post text, post URLs, and post types
- source labels such as Feed, user lists, reply list, or import
- first seen time, last seen time, and local cache update time
- extension settings, language preference, filters, and selection state
- account data from CSV or Markdown files that you choose to import

The extension reads DOM content and page data already rendered by X/Twitter in order to identify accounts, stats, and browsed content.

## How We Use Information

The information above is used only to provide the extension's core features, including:

- showing account stat labels on X/Twitter pages
- collecting and filtering accounts in the local manager page
- importing, exporting, and organizing account lists
- opening profile pages sequentially to refresh missing stats
- saving extension settings and local cache data

## Data Storage

- The extension uses browser local storage to save account cache and settings
- Data is stored locally in your browser environment
- You can clear cached data from the popup
- When the extension is uninstalled, extension data is removed by the browser

## Network Requests

The extension mainly runs on `x.com` and `twitter.com`. When you choose to batch-open profile pages or use sequential refresh, the extension opens or visits the corresponding X/Twitter profile pages in your browser so the page can render current account data.

These requests are sent to X/Twitter itself, not to developer servers. Whether requests include login state, cookies, or other browser credentials depends on your current browser session with X/Twitter and browser rules.

## Data Sharing

- The extension does not upload account cache, imported files, or exported results to developer servers
- The extension does not sell user data
- The extension does not use user data for advertising, profiling, or purposes unrelated to its core features
- The extension does not sync local account lists to a cloud service

## Third-Party Sites

The extension is designed for X/Twitter on the web. X/Twitter's own data handling practices are governed by X/Twitter. The extension cannot control third-party privacy policies, page structure, or access restrictions.

## Accounts And Identity

- The extension does not provide a developer account system
- The extension does not require you to sign in to any developer service
- The extension does not read your X/Twitter password
- The extension does not collect payment information, contacts, precise location, or identity verification information
